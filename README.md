# wallpaper

【壁纸宝贝】是我于2015年开发的一款壁纸类APP，可以通过此款APP查看、下载并设置桌面或者锁屏壁纸。【壁纸宝贝】的设计师是策少，我负责iOS以及Android的开发工作。

 ### 3.参与项目

 - 经历规范、完整的APP开发流程体验，让你对单元测试、多人开发、Code Review、bug管理等有更深入的理解

 - 共享APP各项数据，包括不限于APP的装机量，使用群体，群体特征等

 - 获得收入。我们将按照贡献度分配每位开发者一定的收入
